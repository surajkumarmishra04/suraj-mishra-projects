**BBDU Study Hub** is a modern, premium, and fully responsive web platform designed specifically for BBD University students pursuing B.Tech CSE (Artificial Intelligence). It provides a centralized, easy-to-navigate hub to access semester-wise study materials.

---

## ✨ Features

- **Dark & Light Mode:** Fully functional theme toggle with local storage saving.
- **Modern UI/UX:** Glassmorphism, soft shadows, rounded corners, and a Gen-Z inspired aesthetic.
- **100% Responsive:** Flawless experience across Large Desktops, Laptops, Tablets, and Mobiles.
- **Smooth Animations:** Scroll reveals, button ripples, hover lifts, and page loaders.
- **Modular Codebase:** Clean and separate files for CSS components, responsive media queries, and JS routing.
- **Zero Frameworks:** Built entirely with pure HTML5, CSS3, and Vanilla JavaScript (ES6+) for maximum performance.

## 🚀 Study Resources Provided

- 📚 **Semester-wise Notes:** Well-organized notes for every semester.
- 📝 **Previous Year Question Papers:** Access to past university examination papers.
- 💻 **Assignments:** Practice questions and practical assignments.
- 📖 **Latest Syllabus:** Official semester-wise curriculum.
